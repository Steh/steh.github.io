--- layout: post title: 'Exchange: More than one Active Directory object is configured with the recipient address ...' categories: - Exchange - Exchange 2007 - Exchange 2010 tags: - ADSI Edit - ambigous recipient - english - Event ID 9217 status: publish type: post published: true meta: _syntaxhighlighter_encoded: '1' _edit_last: '1' --- The following error occured often in our "administrativ events" LOG. 
More than one Active Directory object is 
configured with the recipient address IMCAEX-_O=Firma_OU=GRUPPE_cn=Recipients_cn=Steh@lalala.de . 
Messages to this recipient will be deferred 
until the configuration is corrected in Active Directory.
After sending a message to the recipient address, in messagetracking the following error occured 
420 4.2.0 Resolver.ADR.Ambigous; ambigous address

Find the ambigous

Now we have to find out which recipients are ambigous: You have to cut "IMCAEX-" and everything after @ and then replace "_" with "\". 
before:	IMCAEX-_O=Firma_OU=GRUPPE_cn=Recipients_cn=Steh@lalala.de 
after:	\O=Firma\OU=Gruppe\cn=Recipients\cn=Steh
Now you can search in PowerShell: [code lang="powershell" light="true"]Get-Recipient \O=Firma\OU=Gruppe\cn=Recipients\cn=Steh[/code]

Clean UP

Open ADSI Edit an navigate to the User you want to change: Organizational Unit from desired user Now you have to change the usern LegacyExchangeDN to an unique one.
