## Installation

```bash
# Install Docker
sudo apt-get install docker-compose

# Copy OpenCTI Docker Container
mkdir -p /path/to/your/app && cd /path/to/your/app
git clone https://github.com/OpenCTI-Platform/docker.git
cd docker
```
