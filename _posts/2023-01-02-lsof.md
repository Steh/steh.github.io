## Files

```bash
# show all open Files
lsof /var

# suppress open files by kernel
lsof -b

# show the count of all open files
lsof | wc -l

# show open files per user
lsof -u steh

# show all open files under a directory
lsof +D /var/www
```

## Open Networkports

```bash
# show all open ports
lsof -i

# show all open TCP Ports
lsof -i tcp

# find which process is using a specific port
lsof -i :8080
```
