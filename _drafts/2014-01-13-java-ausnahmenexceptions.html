Try und Catch - Fehlerbehandlung in der Methode

Beispiel für Try and Catch: Im Try Block wird die Ausnahmen MyException ausgelöst, der Catch Block fängt diese ab und gibt die Fehlermeldung aus. Alle anderen Fehler sollten im zweiten Block aufgefangen werden.

try{  
   throw new MyException("Fehler aufgetreten...");  
} catch (MyException e) {  
   System.out.print(e.getMessage());  
} catch(Exception e) {  
   System.out.print("Unbekannter Fehler aufgetreten...");  
} [finally] {
  System.out.println("Der Finally Block, ist optional, wird immer ausgeführt");
}

Eigene Ausnahmen

Es ist auch möglich die Ausnahmebehandlung weiter zu reichen. Dazu wird nach den Parametern "throws EXCEPTION" angegeben.

/** Wenn die Ware nicht auf den LKW passt wird eine Ausnahme ausgelöst **/
public void beladen(int ware) throws beladenAusnahme {
  if((ladung + ware) > maxLadung) {
     throw new beladenAusnahme(ware, ladung);
  } else {
     ladung += ware;
  }
}

Entsprechnende Ausnahme:

public class beladenAusnahme extends Exception {
   int ware;
   int ladung;

   public DivisionException(int waree, int ladung) {
      super ("LKW kann nicht beladen werden: " + (ladung + ware));
   }
}
