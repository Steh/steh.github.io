Export ONLY Mailbox:

New-MailboxExportRequest -Mailbox stiki -FilePath '\steh-blog.dec$Exportstiki_mailbox.pst'

Export Mailbox Archive:

New-MailboxExportRequest -Mailbox stiki -FilePath '\steh-blog.dec$Exportstiki_archive.pst' -IsArchive

Export Selected Folders:

Show Foldernames for a Mailbox

Get-MailboxFolderStatistics stiki | Select-Object Name

Create Export Request

New-MailboxExportRequest -Mailbox stiki -IncludeFolders Calendar -FilePath -ExcludeDumpster -ExcludeFolders "Recoverable Items" '\steh-blog.dec$Exportstiki_calendar.pst'
