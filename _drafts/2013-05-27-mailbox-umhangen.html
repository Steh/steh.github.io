  1. Mailbox vom alten User trennen
    2. [code language="Powershell" light="true"]Disable-Mailbox stiki@stiki.de[/code]
  2. Datenbank Updaten
    3. [code language="Powershell" light="true"]Get-MailboxDatabase "stiki-db01" | Clean-MailboxDatabase[/code]
  3. Gelöschte Mailbox suchen
    4. [code language="Powershell" light="true"]Get-MailboxStatistics -server "mbx01" | Where {$_.DisconnectDate -ne $null}[/code]
  4. Mailbox anbinden
    5. [code language="Powershell" light="true"]Connect-Mailbox -Identity 'Mailbox-Guid' -Database 'stiki-db01' -User 'stehson' -Alias 'stehson'[/code]
