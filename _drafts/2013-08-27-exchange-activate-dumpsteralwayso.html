Also such messages that are hard deleted ([SHIFT]+delete).

to do

This Regkey had to be placed on the clients computer.
  1. open regedit and navigate to: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Exchange\Client\Options
  2. new DWORD key, that you name "DumpsterAlwaysOn"
  3. set the numeric value to: 1
  4. restart Outlook
  5. ready
Now you can recover every mail from recover deleted messages menu item
