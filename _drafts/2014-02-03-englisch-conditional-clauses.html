Coditional Clauses

Type 1

Für Dinge die passieren (Wissenschaftlich)
Ist immer wahr 
Conditional (If-) Clause:   Present Tense / Simple Present (I sing)
Main Clause:            will (can, may, must, shall)+infinitive
We will host a party for all supporters if we win the prize.
I will be in Frankfurt when you call me back tomorrow.
If you don't already have the app on your device, you will need to download and install it.
If you click on Rectangle in the documentation page for the Graphics class, you will be taken to the documentation page for the Rectangle class.

Type 2

Dinge die vielleicht passieren können 
Conditional (If-) Clause:   Past Tense (I sang)
Main Clause:            would (could, should)+infinitive
I wouldn't do this job if I didn't like it.
A common programming bug would occur if you forgot to initialize a variable.
If we could rely on sustained funding, we would complete our R&D project by the end of the year.
Everything would be fine if we received their email by the end of the week.
I would tell you if I knew more details.

Type 3

Dienge die nicht passieren werden 
Conditional (If-) Clause:   Past Perfect (I had sung)
Main Clause:            would (could, might, should) have+Past Participle
If we had known more about their plans, we would have cancelled the contract.
If we had got the funding last year, we would have been able to finish the project in March.
We would have told you if we had realized you didn't know.
It would have been worse if the security experts had not reacted to the reports from the management. http://www.multilingual.de/anfanger/zeitformen/
