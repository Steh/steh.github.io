create Shared-Mailbox

create new

New-Mailbox -UserPrincipalName <alias>@<domain> -Alias <alias> -Name <Name> -Database <Database> -OrganizationalUnit <OrganisationalUnit> -Shared[/code]

convert existing Mailbox

Get-Mailbox  | Set-Mailbox -type Shared[/code]

change Rights

Fullaccess

Add-MailboxPermission -Identity "<Mailbox>" -User <User> -AccessRights Fullaccess -InheritanceType all -Automapping $true

Send-As

Add-MailboxPermission -Identity "" -User  -ExtendedRights 'Send-as'

Set Manager

Set-User -IgnoreDefaultScope -Manager "<User" -Identity "<Mailbox>"
