Outlook 2000/2003

  • Open Message in new window by double clicking
  • Select:
    • English: View | Options
    • German: Ansicht | Optionen
  • Go to: Internet Header

Outlook 2007

  • Open Message in new window by double clicking
  • On the Message tab, in the Options group, click the Dialog Box Laucher
  • In the Message Options dialog box, the headers appear in the Internet headers box

Outlook 2010

  • Open Message in new window by double clicking
  • On the Message tab, in the Tags group, click the Dialog Box Laucher
  • In the Message Options dialog box, the headers appear in the Internet headers box

OWA

  • Open Message in new window by double clicking
  • Select Message Options
