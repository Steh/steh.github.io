Give one User FullAccess

You can give only one user FullAccess, this user has now the right to set permissions.
  1. Set the right for one User 
Add-MailboxFolderPermission -Identity "RES-Steh-Car" -User "steh@stiki.de" -AccessRights "FullAccess"
  1. Mount the Calendar in Outlook
  2. right click on the Calendar -> Properties -> Permissions
Set the right for one user 
Add-MailboxFolderPermission -Identity "RES-Steh-Car:Calendar" -User "steh@stiki.de" -AccessRights "ReadItems"
